# NAYA

A calm, mood-aware Flutter UI experience focused on human-centered design.

## Features
- Splash & Intro onboarding
- Mood-based UI flow
- Soft animations
- Clean and readable code structure

## Concept
NAYA invites users to pause, reflect, and gently choose their emotional state through a thoughtful UI journey.

## Tech Stack
- Flutter
- Material Design
